# Cyberzone-Backend

## Follow the following steps to set up your environment

#### create a *`.env`* file in the root directory
***Create the following constants with their values***


- *`PORT`*

- *`DATABASE`*

- *`SECRET_KEY`*

- *`REFRESH_KEY`*

- *`EMAIL_HOST`*

- *`EMAIL_USERNAME`*

- *`EMAIL_PASSWORD`*

In your terminal run the command **`npm run dev`**
